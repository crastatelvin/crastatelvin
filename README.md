## 👨‍💻 About

I ship production-grade AI products. Every project below is deployed, documented, tested, and open to click-through — no tutorials, no forks, no "coming soon". I care about **the last mile**: making sure a model actually reaches a user through a tight UI, a secure backend, and a deploy story that doesn't break on Monday morning.

My work sits at the intersection of **AI systems engineering** (tool-use loops, multi-agent pipelines, RAG), **applied product thinking** (legal, forensics, ops, cybersecurity), and **hardening** (auth, sandboxing, rate limiting, CORS done right).
